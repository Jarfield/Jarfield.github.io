## 使用方式

后续每一章笔记会在开头放：

- 一个系列入口链接
- 一个对应章节原文链接

这样既能统一回到系列页，也能直接打开本章原文。

如果要新建一篇章节笔记，可以直接使用这个模板：

```bash
hexo new deep-learning-book-note "Chapter 1 读书笔记"
```
